# Carpeta Clientes para Compartir

Entrega a cada companero la carpeta `python-client`.

Pasos para cada companero:

1. Abrir carpeta `python-client`.
2. Instalar dependencias.
3. Generar stubs.
4. Ejecutar cliente con la IP del servidor y su nombre.

Ejemplo rapido:

```bash
python client.py --host <IP_DEL_SERVIDOR> --port 50051 --name <NOMBRE>
```

Si el servidor esta en Docker o en otra PC, la IP debe ser la IP LAN real de la maquina servidor.
